🚀 IponHub: Smart Micro-Savings & Rewards Engine
Isang makabagong community platform na binuo para sa mas madali at masayang pag-iipon. Ang IponHub ay pinagsasama ang disiplina sa pagtatabi ng pera at ang saya ng pagtanggap ng rewards sa pamamagitan ng social networking at time-based savings goals.

📖 Overview
Ang proyektong ito ay isang scalable system para sa mga digital saving circles. Ang mga users ay maaaring magsimula bilang Free Ambassadors o mag-level up bilang Certified Iponers para ma-enjoy ang mga "Saving Cycles" na may kasamang bonuses. Lahat ng rewards ay ibinibigay sa anyo ng Digital Credits na may 1:1 value sa Philippine Peso (PHP).

👥 Membership Tiers
Ang system ay gumagamit ng dalawang level ng access para mapanatiling balanse ang rewards pool:

1. Starter Member (Free)
Access: 100% Free Registration.

Perks: Magkakaroon ng sariling Referral Code. Kumita ng credits sa bawat kaibigang sasali.

Limitation: Ang credits ay maaaring gamitin sa loob ng app (tulad ng Mobile Load) o i-store muna hanggang sa mag-upgrade ang account.

2. VIP Saver (Plus)
Access: Activation fee na 100 PHP (magsisilbing base balance).

Perks: * Full access sa reward system at unlimited withdrawals.

Access sa "Ipon Boosters" menu para sa mabilisang paglago ng credits.

Priority processing sa lahat ng claims.

📜 Ipon Boosters (Saving Cycles)
Kapag ang user ay isa nang VIP Saver, maaari na nilang ilagay ang kanilang credits sa mga "Time-Locked Challenges."

Halimbawa: 7-Day Quick Growth Challenge

Duration: 7 Araw

Milestone Reward: 20% Bonus na papasok sa dashboard matapos ang challenge.

Control: Pagkatapos ng 7 araw, pwedeng i-withdraw ang lahat (Base + Bonus) o i-restart ang challenge para mas lumaki pa ang rewards.

💰 The Credit Economy
Fixed Valuation: 1 System Credit = 1 PHP.

Utility: * Claim Rewards: I-withdraw ang credits diretso sa iyong Bank o E-Wallet (Exclusive for VIPs).

Mobile Load: Built-in gateway para gawing prepaid load ang credits (Available sa lahat ng networks).

Cycle Restart: Gamitin ang naipong credits para sumali sa mas malalaking Ipon Challenges.

⚙️ System Features
✅ Smart Buddy Tracking: Automated na pag-monitor ng invites at rewards distribution.
✅ Tiered Access Control: Maayos na paghihiwalay ng Starter at VIP features.
✅ Time-Locked Logic: Safe at secure na countdown para sa mga saving milestones.
✅ 1:1 Reward System: Madaling intindihin na conversion (Piso-to-Piso).
✅ Load API Ready: Instant conversion mula credits tungo sa Mobile Load.
